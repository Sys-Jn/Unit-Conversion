# 🖥️ Miles to Kilometers Converter (Tkinter GUI)

A simple **GUI-based Unit Converter** built using **Python Tkinter**.
This application allows users to convert distance from **miles to kilometers** through an interactive interface.

---

## 🚀 Features

* User-friendly graphical interface (GUI)
* Convert miles to kilometers instantly
* Clean and minimal design
* Real-time result display

---

## 🛠️ Tech Stack

* **Python**
* **Tkinter (GUI Library)**

Concepts used:

* GUI Development
* Event Handling
* Functions & Variables

## 📌 Conversion Formula

Kilometers = Miles × 1.60934
